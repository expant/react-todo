# React Todos

### 🛠️ Technology Stack

- TypeScript
- React
- Vite

### ⚙️ Installation

```bash
git clone https://github.com/expant/react-todo.git
cd react-todo
npm install
```

### 💻 Development

```bash
npm run dev
```

### ✅ Tests

```bash
npm run test
```

### 🏗️ Production Build

```bash
npm run build
npm run preview
```
